# 🏪 Ndokoti Stock — Application de Gestion de Stock

Application backend développée avec **Spring Boot 3** pour la gestion du stock
de l'épicerie **Ndokoti Shop**.

## Fonctionnalités
- Gestion des produits (CRUD)
- Suivi du stock en temps réel
- Alertes de stock bas
- Gestion des commandes fournisseurs
- Mouvements d'entrée/sortie
- Authentification sécurisée (JWT + Spring Security)
- Rôles : ADMIN, EMPLOYE

## Stack technique
| Couche | Technologie |
|--------|-------------|
| Backend | Java 17 + Spring Boot 3.2 |
| Sécurité | Spring Security + JWT |
| Base de données | PostgreSQL + Spring Data JPA |
| Build | Maven |
| Frontend (à venir) | Angular |
